# 学生成绩管理系统 — 系统使用说明

## 一、系统概述

学生成绩管理系统是一个面向学生、教师、院长和管理员的综合性成绩管理平台，支持批量数据导入、成绩管理、试卷分析等功能。

---

## 二、用户登录

### 2.1 登录方式

1. 打开系统登录页面
2. 输入用户ID（支持数字和字符串，如：A0001、T0001、S230001）
3. 输入密码
4. 选择用户类型（学生/老师/admin）
5. 点击"登陆"按钮

### 2.2 用户类型说明

- **学生**：使用学号登录（如：S230001）
- **教师**：使用教师工号或教师ID登录（如：T0001 或 1）
- **管理员**：使用管理员工号登录（如：A0001）

---

## 三、管理员功能

### 3.1 学生管理

#### 3.1.1 添加学生
- 路径：学生管理 → 添加学生
- 功能：单个添加学生信息
- 字段：学生姓名、初始密码

#### 3.1.2 学生批量导入
- 路径：学生管理 → 学生批量导入
- 功能：通过Excel批量导入学生
- 模板下载：点击"下载模板"按钮
- 模板格式：
  - 列1：学生姓名
  - 列2：初始密码
  - 列3：班级ID
  - 列4：年级
  - 列5：专业ID
  - 列6：学院ID

#### 3.1.3 学生列表
- 路径：学生管理 → 学生列表
- 显示信息：学号、姓名、密码、年级、班级ID、专业ID、学院ID
- 操作：编辑、删除

### 3.2 教师管理

#### 3.2.1 教师批量导入
- 路径：教师管理 → 教师批量导入
- 功能：通过Excel批量导入教师
- 模板下载：点击"下载模板"按钮
- 模板格式：
  - 列1：教师工号
  - 列2：教师姓名
  - 列3：初始密码
  - 列4：角色(admin/teacher/dean)
  - 列5：所属学院ID

### 3.3 课程管理

#### 3.3.1 课程批量导入
- 路径：课程管理 → 课程批量导入
- 功能：通过Excel批量导入课程
- 模板下载：点击"下载模板"按钮
- 模板格式：
  - 列1：课程名称
  - 列2：学分
  - 列3：所属学院ID
  - 列4：所属专业ID

### 3.4 开课表管理

#### 3.4.1 开课管理
- 路径：开课表管理 → 开课管理
- 功能：查询和管理开课信息

#### 3.4.2 批量导入
- 路径：开课表管理 → 批量导入
- 功能：通过Excel批量导入开课表
- 模板下载：点击"下载模板"按钮
- 模板格式：
  - 列1：课程ID
  - 列2：教师ID
  - 列3：开课学期

### 3.5 学院/专业/班级管理

#### 3.5.1 批量导入
- 学院管理 → 批量导入：模板包含"学院名称"
- 专业管理 → 批量导入：模板包含"专业名称、所属学院ID"
- 班级管理 → 批量导入：模板包含"班级名称、年级、专业ID、学院ID"
- 所有页面均提供模板下载功能

### 3.6 成绩管理

#### 3.6.1 成绩上传
- 路径：成绩管理 → 成绩上传
- 功能：批量上传成绩单
- 模板下载：点击"下载成绩单模板"按钮
- 模板格式要求：
  - 第2行：课程元信息（课程名称、课程类别、课程性质、考核方式、任课教师）
  - 第3行：教学信息（开课学期、学分、学时、开课班级、年级）
  - 第4行：表头（A-G为第一个学生，H-N为第二个学生）
  - 第5-73行：学生数据

#### 3.6.2 成绩查询与修改
- 路径：成绩管理 → 学生成绩查询
- 功能：查询和修改学生成绩（仅管理员可修改）

### 3.7 试卷分析

#### 3.7.1 试卷分析管理
- 路径：试卷分析 → 文件列表
- 功能：上传、查看、修改、删除Word格式的试卷分析文档
- 上传格式：支持.doc和.docx文件

## 七、常见问题

### 7.1 登录问题
- **问题**：提示"id is not a number"
- **解决**：系统已支持字符串ID，如A0001、T0001等，直接输入即可

### 7.2 模板下载问题
- **问题**：无法下载模板
- **解决**：检查网络连接，确保浏览器允许下载文件

### 7.3 文件上传问题
- **问题**：上传失败
- **解决**：
  1. 检查文件格式是否正确（.xlsx或.xls）
  2. 检查文件内容是否符合模板要求
  3. 查看错误提示信息

### 7.4 权限问题
- **问题**：无法执行某些操作
- **解决**：确认当前登录账户的权限，部分功能仅管理员可用

---

## 八、注意事项

1. **数据备份**：批量导入前建议备份数据
2. **模板格式**：严格按照模板格式填写，避免导入失败
3. **权限控制**：教师不能修改成绩，只能通过上传方式录入
4. **文件大小**：上传文件大小限制为50MB
5. **成绩格式**：成绩单模板必须严格按照指定格式，否则可能解析失败
